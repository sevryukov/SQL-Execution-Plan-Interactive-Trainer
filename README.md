# SQL Plan Trainer — Тренажёр планов выполнения SQL

Интерактивный тренажёр для изучения планов выполнения SQL-запросов. Работает полностью в браузере, без серверной части.

## 📖 Описание

**SQL Plan Trainer** — это микро-обучающая платформа, которая помогает понять, как СУБД выбирает операторы для выполнения запросов. Приложение предоставляет:

- Теоретические материалы по различным типам соединений и операторов
- Интерактивные практические задания с мгновенной обратной связью
- Систему очков для мотивации обучения
- Поддержку двух языков: русского и английского

## 🎯 Возможности

- **Теоретическая панель** — изучайте концепции планов выполнения с наглядными схемами
- **Практическая панель** — отвечайте на вопросы по темам и получайте обратную связь
- **Навигация по темам** — переключайтесь между модулями обучения
- **Сохранение прогресса** — ваши очки и прогресс сохраняются в localStorage
- **Двуязычный интерфейс** — переключение между русским и английским языками

## 🚀 Быстрый старт

### Локальный запуск

1. Клонируйте репозиторий или скачайте файлы проекта
2. Откройте `index.html` в любом современном браузере

Или используйте локальный веб-сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (если установлен npx)
npx serve .

# PHP
php -S localhost:8000
```

Затем откройте в браузере: `http://localhost:8000`

### Развёртывание на GitHub Pages

1. Загрузите файлы в репозиторий GitHub
2. Включите GitHub Pages в настройках репозитория
3. Приложение будет доступно по адресу: `https://ваш-логин.github.io/имя-репозитория/`

## 🌐 Мультиязычность

Приложение поддерживает два языка интерфейса:
- 🇬🇧 English
- 🇷🇺 Русский

Переключение языка осуществляется через выпадающий список в верхней панели. Контент (теория и задания) загружается из соответствующей папки (`content/en/` или `content/ru/`).

## 📝 Формат данных

### Манифест тем (manifest.json)

```json
{
  "topics": [
    {
      "id": "module1",
      "title": "Название темы",
      "theoryFile": "module1.md",
      "taskFile": "module1.json"
    }
  ]
}
```

### Практическое задание (task JSON)

```json
{
  "diagramTitle": "Заголовок диаграммы",
  "diagram": "Текст диаграммы",
  "prompt": "Вопрос пользователю",
  "options": ["Вариант 1", "Вариант 2", ...],
  "correctIndex": 1,
  "points": 10,
  "feedback": {
    "correct": "Сообщение при правильном ответе",
    "incorrect": "Сообщение при неправильном ответе"
  }
}
```

## 🔧 Расширение контента

Чтобы добавить новые темы:

1. Создайте файлы теории в `content/{lang}/theory/` (формат Markdown)
2. Создайте файлы заданий в `content/{lang}/tasks/` (формат JSON)
3. Обновите `manifest.json`, добавив новую тему
