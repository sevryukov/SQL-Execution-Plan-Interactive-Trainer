# SQL Plan Trainer 🚀

Образовательное веб-приложение для изучения планов выполнения SQL запросов.

## 📖 Описание

**SQL Plan Trainer** - это интерактивная платформа, предназначенная для обучения специалистов работе с планами выполнения SQL запросов в СУБД Oracle. Приложение помогает понять, как база данных обрабатывает запросы, и научиться оптимизировать их производительность.

## ✨ Основные возможности

- 🌐 **Двуязычная поддержка** - русский и английский языки интерфейса
- 📊 **Отслеживание прогресса** - автоматическое сохранение результатов обучения
- 🎯 **Интерактивные задания** - практические упражнения с мгновенной проверкой
- 📝 **Теоретические материалы** - подробные объяснения концепций
- 🏆 **Система достижений** - мотивация через геймификацию
- 📱 **Адаптивный дизайн** - работа на любых устройствах

## 🚀 Быстрый старт

### Требования

- Любой современный веб-браузер (Chrome, Firefox, Safari, Edge)
- Локальный веб-сервер (опционально, но рекомендуется)

### Установка и запуск

1. **Клонируйте репозиторий:**
```bash
git clone <url-репозитория>
cd sql-plan-trainer
```

2. **Запустите локальный веб-сервер:**

   **Вариант 1: Python 3**
   ```bash
   python3 -m http.server 8000
   ```

   **Вариант 2: Node.js (http-server)**
   ```bash
   npx http-server -p 8000
   ```

   **Вариант 3: PHP**
   ```bash
   php -S localhost:8000
   ```

3. **Откройте приложение в браузере:**
   ```
   http://localhost:8000
   ```

### Без сервера

Приложение также работает при прямом открытии файла `index.html` в браузере, но для корректной работы некоторых функций рекомендуется использовать веб-сервер.

## 📚 Как добавить новый контент

### Добавление темы

Откройте файл `data.js` и добавьте новую тему в объект `topics`:

```javascript
topics: {
  existingTopic: { ... },
  newTopic: {
    order: 3,
    en: "New Topic",
    ru: "Новая тема"
  }
}
```

### Добавление теоретического материала

В файле `data.js` добавьте запись в `theoryData`:

```javascript
theoryData: {
  existingTheory: { ... },
  newTheory: {
    topic: "newTopic",
    order: 1,
    en: {
      title: "Theory Title",
      content: "**Markdown** содержание..."
    },
    ru: {
      title: "Название теории",
      content: "**Markdown** содержание..."
    }
  }
}
```

### Добавление практического задания

В файле `data.js` добавьте упражнение в `exercises`:

```javascript
exercises: {
  existingExercise: { ... },
  newExercise: {
    topic: "newTopic",
    order: 1,
    en: {
      question: "Question text?",
      options: ["Option 1", "Option 2", "Option 3"],
      correctAnswer: 0,
      explanation: "Explanation text."
    },
    ru: {
      question: "Текст вопроса?",
      options: ["Вариант 1", "Вариант 2", "Вариант 3"],
      correctAnswer: 0,
      explanation: "Текст объяснения."
    }
  }
}
```

## 🌐 Поддержка языков

Приложение поддерживает два языка:

- **🇷🇺 Русский** (по умолчанию)
- **🇬🇧 English**

Переключение языка осуществляется через кнопку в верхней панели приложения. Выбранный язык сохраняется в `localStorage` и используется при следующих посещениях.

Для добавления нового языка:

1. Добавьте переводы в `localization.js`
2. Добавьте переводы тем, теории и упражнений в `data.js`
3. Обновите функцию `updateLanguage()` в `script.js`

## 💾 Хранение состояния

Приложение использует `localStorage` для сохранения:

- Прогресса прохождения тем
- Результатов выполнения упражнений
- Предпочтений пользователя (язык, настройки)
- Статистики обучения

Структура ключей:
- `sqlTrainer_currentLang` - текущий язык
- `sqlTrainer_completedExercises` - выполненные упражнения
- `sqlTrainer_userProgress` - общий прогресс пользователя

## 🔍 Как это работает

1. Пользователь выбирает тему из списка
2. Изучает теоретический материал (с поддержкой Markdown)
3. Выполняет практические задания
4. Получает мгновенную обратную связь
5. Прогресс автоматически сохраняется
6. Может отслеживать свой прогресс в личном кабинете
